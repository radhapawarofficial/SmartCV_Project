# SmartCV - ATS-Friendly Resume Builder

<strong>SmartCV</strong> is a web-based application developed as a final-year project to help job seekers create professional, ATS-optimized resumes. By providing structured inputs and real-time guidance, it ensures that your resume bypasses automated screening software and reaches the recruiter's desk.

## 🚀 Live Demo
Check out the live website: https://smartcv-ats-friendly-resume-builder.onrender.com

## 📖 Project Overview
This project was developed to address the challenges job seekers face with Applicant Tracking Systems (ATS). Instead of just generating a document, <strong>SmartCV</strong> focuses on content optimization and structured data.

## 📑 Key Pages & Features

*   <strong>Home Page:</strong> The main landing page showcasing the value proposition of ATS-friendly resumes.
*   <strong>About Page:</strong> Detailed information regarding the mission and vision of the <strong>SmartCV</strong> project.
*   <strong>Sign Up / Login Page:</strong> Secure user authentication system to manage personal profiles.
*   <strong>Dashboard:</strong> A personalized hub where users manage their resume drafts and profile settings.
*   <strong>Create CV Page:</strong> The core feature! An intuitive form to input career history, education, and skills.
*   <strong>Testimonial Page:</strong> A community space where users share their experience and feedback.
*   <strong>Footer:</strong> Quick navigation and contact links.

## 🛠️ Tech Stack
*   <strong>Framework:</strong> Django (Python)
*   <strong>Frontend:</strong> HTML5, CSS3, JavaScript
*   <strong>Styling:</strong> Bootstrap 5, Tailwind CSS
*   <strong>Database:</strong> SQLite3
*   <strong>Version Control:</strong> Git & GitHub
